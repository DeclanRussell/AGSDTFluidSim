Detailed Description

our cuda libraries

This class uses cuda and parralel algorithms to manage particles in a fluid simulation.

our SPH kernals

Author
Declan Russell
Date
09/03/2015
Version
1.0 Our fluid simulation uses smoothed particle hydrodynamics and true navier stokes equations to compute our particle positions in our simulation. This class also manages its own openGL buffer storing postions of particles in vertex attrip pointer 0. In case you wish to draw these particles.
